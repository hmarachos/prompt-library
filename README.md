# Prompt Library

Репозиторий содержит проверенные командные промпты для корпоративных и продуктовых задач.
Все промпты оформлены как Library Assets с версионированием, владельцами и тест-кейсами.

## Принципы библиотеки
- Повторяемость результата
- Минимизация скрытого контекста
- Жёсткие правила для критичных данных
- Поддержка командной работы и ревью

## Структура
Каждый промпт хранится в отдельной директории и содержит:
- `README.md` — карточку промпта
- `prompt_vX.X.txt` — актуальную версию шаблона
- `tests.md` — тест-кейсы и ожидания

## Правила изменений
- Любые изменения → новая версия
- У версии есть владелец
- Изменения описываются в PR
