## Назначение
Генерация корпоративных поздравлений с днём рождения сотрудников проектной организации
с гарантией корректности ФИО, должностей и структуры текста.

## Контекст использования
- HR-коммуникации
- Внутренние порталы
- Корпоративные рассылки
- Ассистенты руководителей

## Ключевые требования
- ФИО обрабатывается как неизменяемый идентификатор
- Склоняется только подразделение
- Должность не изменяется
- Обращение определяется строго по полу из входных данных
- Структура вывода фиксированная

## Основные риски
- Склонение или изменение ФИО
- Нарушение структуры текста
- Ошибки согласования

## Механизмы защиты
- Жёсткие текстовые правила
- Явная финальная самопроверка
- Формализация входных данных
