## [1.0.0] — 2026-02-09
### Added
- Первый production-ready командный промпт `Birthday_Congratulations_Corporate`
- Жёсткие правила обработки ФИО как неизменяемого идентификатора
- Фиксированная структура вывода поздравления
- Поддержка гендерных форм обращений
- Явный блок финальной самопроверки перед выводом

### Documentation
- Карточка промпта (`birthday_congratulations/README.md`)
- Описание библиотеки (`prompt-library/README.md`)
- Тест-кейсы с ожидаемым поведением (`tests.md`)
- Рекомендованная структура репозитория

### Security
- Исключено склонение и автокоррекция ФИО
- Исключены неявные изменения должностей
- Минимизирован риск грамматических ошибок в официальных коммуникациях

## Versioning Notes
- MAJOR — изменения, ломающие обратную совместимость
- MINOR — расширение функциональности без нарушения текущего API промпта
- PATCH — исправления формулировок и документации без изменения поведения
